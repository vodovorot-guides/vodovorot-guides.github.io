# Способности Шамана Исцеления

## Основные способности

• {{ site.data.spells.riptide }} – Мгновенно исцеляет союзника и накладывает на него эффект периодического исцеления на **18** секунд. Одно из самых эффективных заклинаний, в первую очередь за счет первоначального исцеления. Время восстановления **6** секунд. Дает один заряд {{ site.data.spells.tw22 }}.

> Крайне сильное заклинание мгновенного исцеления и наш основной ХоТ. Рекомендуется использовать по кулдауну и всегда держать на активном танке.

• {{ site.data.spells.hs }} – Быстрое, но дорогое по мане исцеление. Поглощает заряд {{ site.data.spells.tw22 }}, увеличивая шанс критического удара на **40%**.

> Наша «флешка» – много хилит и дорого стоит. Если спамить этим заклинанием, то можно очень быстро остаться без маны. Лучший кандидат для поглощения стаков {{ site.data.spells.tw22 }}.

• {{ site.data.spells.hw }} – Медленное, но эффективное по мане исцеление. Поглощает заряд {{ site.data.spells.tw22 }}, ускоряя произнесение на **20%**.

> Наш исцеляющий «филлер». Хилит сильнее чем {{ site.data.spells.hs }}, но дольше кастуется.

• {{ site.data.spells.ch }} – Исцеляет союзника на средние значения, и затем перескакивает на три дополнительные цели, с каждым разом уменьшая количество лечения на **30%**. Наше основное заклинание для АоЕ отхила. Для перескока всегда выбирается цель с минимальным количеством здоровья. Дальность прыжка – **15** метров. Дает один заряд {{ site.data.spells.tw22 }}. 

> Наше основное заклинание для АоЕ отхила. Подробнее про механику работы этой способности [в конце раздела](https://stormkeeper.ru/resto/rotation.html#%D1%86%D0%B5%D0%BF%D0%BD%D0%BE%D0%B5-%D0%B8%D1%81%D1%86%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5).

• {{ site.data.spells.rain }} – Целительные ливни заливают указанную область в радиусе **10** метров, восстанавливая здоровье шести союзникам в течение **10** секунд. Время восстановления **10** секунд. Эффективное заклинание для лечения группы союзников.

> Наш короткий АоЕ-кулдаун, который следует использовать на максимальное скопление игроков. 

• {{ site.data.spells.hst }} – Вы призываете рядом с собой тотем на **15** секунд., который раз в **2** секунды восполняет здоровье одному раненому союзнику.  Время восстановления **30** секунд. 

> Можно ставить практически по кулдауну. Заменяется на {{ site.data.spells.cbt }} при взятии этого таланта.

• {{ site.data.spells.water_shield }} – Вас окружает щит из водяных сфер, раз в **5** секунд восполняющий **50** ед. маны. При получении урона в ближнем бою щит восполняет вам **2%** маны. Имеет **3** заряда на получение урона, после потери которых щит следует обновлять.

> Наш основной стихийный щит для поддержания. Должен всегда висеть на вас во время боя. Невозможно использовать одновременно с {{ site.data.spells.ls }} или {{ site.data.spells.eshield }}.

• {{ site.data.spells.eshield }} – Окружает союзника щитом земли, который усиливает исцеление применяемое вами к этому союзнику на **20%** и восполняет ему здоровье каждый раз, когда он получает урон (не чаще одного раза в **3** секунды). Максимум **9** зарядов.

> Данный щит всегда должен висеть на одном из союзников, желательно на активном танке. Обновляйте по спадению стаков.

## Атакующие способности

• {{ site.data.spells.fs }} – Поджигает противника, нанося урон от огня и накладывая ДоТ эффект на **18** секунд. Время восстановления **6** секунд.

> Наш основной дамажащий ДоТ, который желательно поддерживать на боссе и на самых долгоживущих мобах. Дает проки {{ site.data.spells.lava_surge }}, что увеличивает урон еще больше.

• {{ site.data.spells.lvb }} – Запускает в цель жидкую лаву, нанося урон от огня. {{ site.data.spells.lvb }} всегда наносит критический урон, если цель поражена заклинанием {{ site.data.spells.fs }}. Время восстановления **8** секунд.

> Наше самое дамажащее заклинание в бою против одной цели. Кастуется по кд и особенно по проку {{ site.data.spells.lava_surge }}, в свободное от лечение время.

• {{ site.data.spells.lb }} – Запускает в цель молнию, нанося урон от сил природы.   

> Филлер для нанесения урона, когда {{ site.data.spells.lvb }} на кулдауне.

• {{ site.data.spells.cl }} – Поражает противника молнией, которая наносит урон от сил природы и затем отскакивает в других находящихся рядом противников. Максимум поражает **3** цели.

> Основное заклинание для АоЕ урона, хоть и поражает всего 3 цели.

• {{ site.data.spells.frs }} – Замораживает цель, нанося урон от магии льда и снижая её скорость передвижения на **50%** на **6** секунд. Мгновенное заклинание, может применяться на ходу. 

> Пригодится для нанесения урона на ходу или как замедление одной цели.

• {{ site.data.spells.primal }} – Бесполезная способность для Шамана Исцеления.

• {{ site.data.spells.ftw }} – Бесполезная способность для Шамана Исцеления.

• {{ site.data.spells.ls }} – На **30** минут окружает вас щитом молний, который наносит урон от сил природы атакующим вас в ближнем бою противникам. Максимум можно иметь только один щит стихий, и для Шамана Исцеления лучшим выбором будет {{ site.data.spells.water_shield }}.

> Практически не используется, так как {{ site.data.spells.water_shield }} для рестора лучше.

## Пассивные способности

• {{ site.data.spells.lava_surge }} – Периодический урон от {{ site.data.spells.fs }} имеет шанс восстановить {{ site.data.spells.lvb }} и сделать следующее его применение мгновенным.  

> Этот прок – одна из основных причин поддерживать {{ site.data.spells.fs }} на всех целях.

• {{ site.data.spells.rez }} – Позволяет воскреснуть в случае смерти. Не считается боевым воскрешением, имеет собственное время восстановления.

> Уникальная способность, которая есть только у шамана. Долгое время восстановления не позволит строить тактику на боссе вокруг этой способности, но может иметь решающее значение на удачном пуле в рейде или Мифик+.

• {{ site.data.spells.resurgence }} – Критические эффекты заклинаний прямого исцеления восполняют некоторое количество маны.  

> Один из источников восстановления маны для рестора. Немного повышает ценность Критического удара, но не дайте себя обмануть – Крит хорош и без этой пассивки, это просто приятное дополнение.

• {{ site.data.spells.tw }} – Применяя {{ site.data.spells.ch }} и {{ site.data.spells.riptide }} вы сокращаете время применения следующей {{ site.data.spells.hw }} на **20%** либо повышаете вероятность критического эффекта {{ site.data.spells.hs }} на **40%**. Суммируется до **2** раз.  

> Трата проков на {{ site.data.spells.hs }} дает больше исцеления, чем на {{ site.data.spells.hw }}. У нас не стоит задача потратить все эти проки, нет ничего страшного если они будут уходить в оверкап. 

## Мощные способности, «Бурсты»

• {{ site.data.spells.htt }} – каждые **2** секунды производит импульсы, которые исцеляют всех союзников в радиусе **40** метров от тотема. В это время Шаман может продолжать использовать другие заклинания. 

Используйте эту способность в те моменты, когда рейд получает большое количество урона или если множество союзников имеют малое количество здоровья. Во время установки тотема мгновенно проходит один импульс исцеления, поэтому не стоит ставить его до получения урона, во избежание переисцеления. Этот тотем также полезен во время активного передвижения.

• {{ site.data.spells.slt }} – уникальная способность, аналогов которой нет ни у одного другого класса. Вы призываете в указанном месте тотем, который в радиусе **10** метров снижает весь получаемый союзниками урон на **10%** и перераспределяет запас здоровья целей так, чтобы все они имели одинаковый процент максимального запаса здоровья.

Это очень мощная способность, которая способна спасти множество союзников, особенно в сочетании с талантом {{ site.data.spells.asc_resto }} и другими АоЕ заклинаниями. Используйте {{ site.data.spells.slt }} когда рейд стоит кучно и получает большое количество урона, как разового, так и растянутого по времени.

В Мифик+ этот тотем может использоваться для спасения жизни танка, мгновенно подняв его здоровье за счет союзников.

• {{ site.data.spells.slt }} очень хорошо сочетается со способностью Охотников на демонов – {{ site.data.spells.darknessdh }}, полностью убирая его случайный эффект и превращая эту способность в **20%** снижение всего урона для рейда.


## Защитные способности

• {{ site.data.spells.as }} – наша единственная базовая защитная способность. Снижает весь получаемый урон на **40%** на **12** секунд и не имеет ГКД.

Учитывая короткий кулдаун в полторы минуты, эту способность можно использовать достаточно часто перед получением урона. Достаточно долгое время действия позволяет снижать урон от различных ДоТ заклинаний и поможет пережить перебежки, когда способность хилить ограничена.

## Вспомогательные способности

Шаман Исцеления обладает большим количеством различных **«Utility»** заклинаний, которые будут полезны в рейде и подземельях.

• {{ site.data.spells.bl }} / {{ site.data.spells.hero }} – Повышает скорость атак и произнесения заклинаний союзников на **30%** на **40** сек. Союзники, получившие этот эффект, испытывают чувство насыщения и не могут повторно получить подобный эффект.

Используйте это заклинание с умом, предварительно скоординировав его активацию с Рейд Лидером или вашей группой. 
 
• {{ site.data.spells.swg }} – Позволяет вам применять заклинания на ходу в течение **15** секунд. Время восстановления **2** минуты.

Не имеет ГКД и эту способность можно использовать во время произнесения другого заклинания. 

• {{ site.data.spells.mtt }} – Вы призываете рядом с собой тотем на **8** секунд, который увеличивает регенерацию маны на **100%** всем союзникам в радиусе **20** метров. Время восстановления **3** минуты.

Координируйте установку этого тотема с другими целителями, так как радиус его действия достаточно маленький.
 
• {{ site.data.spells.purge }} – Снимает с противника один благотворный магический эффект.

Крайне полезное заклинание в ПвП, а также находит применение в ПвЕ в различных подземельях. Помимо Шамана, аналоги такого заклинания есть у Жреца, Охотника на демонов, Мага, Охотника (питомец) и Чернокнижника (питомец).

• {{ site.data.spells.ws }} – Прерывает применяемое противником заклинание.

Имеет самое низкое время восстановления в игре среди всех интераптов – **12** секунд. К тому же Шаман Исцеления единственный хилер с полноценным прерыванием. Это заклинание имеет дальность **30** метров, в отличие от остальных наших атакующих заклинаний (**40** метров). 
 
• {{ site.data.spells.wolf }} – Вы превращаетесь в призрачного волка. Ваша скорость передвижения повышается на **30%** и не может быть снижена менее чем до **100%**.

Полезно для быстрого перемещения во время боя, но старайтесь проводить в форме волка как можно меньше времени, так как в это время не можете произносить заклинания.

• {{ site.data.spells.dispel }}– Снимает все магические эффекты и проклятия с выбранного союзника. Время восстановления **8** секунд, но только в том случае, если произошло рассеивание эффекта, иначе время восстановления не запускается.

• {{ site.data.spells.hex }} – Превращает выбранного гуманоида или зверя в лягушку на **1** минуту. Время восстановления **20** секунд.

Под действием этой способности противник может свободно перемещаться, но не может атаковать или применять заклинания. Получаемый целью урон может прервать действие эффекта, но это происходит не сразу. 

• {{ site.data.spells.ebt }} –  Вы призываете в указанном месте тотем на **20** секунд, который замедляет противников в радиусе **10** метров вокруг себя на **50%**. Время восстановления **30** секунд.

• {{ site.data.spells.ct }} – Вы призываете в указанном месте тотем, который спустя **2** сек. после установки оглушает противников в радиусе **8** вокруг себя на **3** секунды. Время восстановления **1** минута.

• {{ site.data.spells.tremor }} – Вы призываете рядом с собой тотем на **10** секунд, который сотрясает землю вокруг себя, снимая с союзников эффекты страха, подчинения и сна. Время восстановления **1** минута.

Все эффекты снимаются раз в секунду, начиная с момента установки тотема. Невозможно применить, будучи под действием вышеперечисленных эффектов, так что тотем следует ставить заранее. 

• {{ site.data.spells.igor }} – Призывает большого элементаля земли, защищающего вас или ваших союзников в течение **1** мин. Время восстановления **5** минут.

Не таунтит боссов, но если танк умер, то элементаль может продержать на себе мобов некоторое время.
 
• {{ site.data.spells.mass_res }} – Воскрешает всех союзников вне боя.  

• {{ site.data.spells.waterwalking }}  – Позволяет союзнику ходить по воде. Полученный урон снимает эффект.

Применение маунта под действием этого баффа позволяет ходить по воде и на маунте. Не имеет ГКД. Полезно для открытого мира и некоторых подземелий.

• {{ site.data.spells.far }} – Переносит точку обзора в указанное место.

Крайне полезное заклинание в Торгасте, так как позволяет заранее узнать расположение всех сокровищ. 

• {{ site.data.spells.recall }} – Дополнительный Камень возвращения, возвращает вас в привязанную локацию.  

> Другие вспомогательные способности доступны в талантах, их описание можно найти в [этом разделе](https://stormkeeper.ru/resto/talents.html).

# Приоритет заклинаний в рейде

1. Поддерживайте {{ site.data.spells.eshield }}, желательно на танке.
2. Поддерживайте на себе {{ site.data.spells.water_shield }}.
3. Используйте {{ site.data.spells.riptide }} по кулдауну и поддерживайте ХоТ на активном танке.
4. Используйте {{ site.data.spells.hst }} / {{ site.data.spells.cbt }} (в зависимости от выбранного таланта) так часто, как это возможно. Повторное применение {{ site.data.spells.cbt }} высвободит накопленное исцеление. Используйте это чтобы спасти жизнь союзников в нужный момент или чтобы предотвратить избыточное исцеление от тотема.
5. Используйте {{ site.data.spells.ul }} по кулдауну. Лучшие заклинания для усиления это {{ site.data.spells.ch }} – если вам необходимо исцеление по области, либо {{ site.data.spells.riptide }} / {{ site.data.spells.hw }} / {{ site.data.spells.hs }} для лечения одной цели.
6. Используйте {{ site.data.spells.rain }} таким образом, чтобы задеть максимальное количество раненых союзников, например применяйте эту способность в зону ближнего боя вокруг босса.
7. Используйте {{ site.data.spells.ewt }} по той же схеме, что и {{ site.data.spells.rain }}. 
8. Если необходимо исцеление по области, используйте {{ site.data.spells.ch }} в того союзника, который находится близко к остальным раненым. Используйте {{ site.data.spells.downpour }} и {{ site.data.spells.wellspring }} по кулдауну, если они взяты в талантах.
9. Тратьте заряды {{ site.data.spells.tw22 }} для исцеления одной цели – {{ site.data.spells.hs }} для быстрого отхила, либо {{ site.data.spells.hw }} как более эффективный по мане вариант.
10. Кастуйте {{ site.data.spells.hw }} в качестве эффективного по мане заклинания, даже без стаков {{ site.data.spells.tw22 }}.
11. Используйте {{ site.data.spells.slt }} перед получением большого количества АоЕ урона, а после его получения – {{ site.data.spells.htt }}.
12. Если нечего хилить – кастуйте {{ site.data.spells.fs }} и тратьте проки {{ site.data.spells.lava_surge }} на {{ site.data.spells.lvb }}. Не забывайте, что эти заклинания стоят ману и распоряжаться ей надо с умом.
13. Если {{ site.data.spells.lvb }} на кулдауне, а хилить все еще нечего, используйте {{ site.data.spells.lb }} или {{ site.data.spells.cl }}.

> За счет Искусности {{ site.data.spells.resto_mastery }}, всегда старайтесь выбирать для ваших заклинаний наиболее раненую цель. Обращайте внимание на расположение союзников, так как это очень важно для эффективного использования {{ site.data.spells.ch }}.

# Приоритет заклинаний в Мифик+

В подземельях приоритет лечения смещается с АоЕ отхила от {{ site.data.spells.ch }} и {{ site.data.spells.rain }} на исцеление одной цели при помощи {{ site.data.spells.hw }}, {{ site.data.spells.hs }} и {{ site.data.spells.riptide }}.

{{ site.data.spells.hw }} имеет достаточно долгое время произнесения, поэтому старайтесь усилять его с помощью {{ site.data.spells.tw22 }} или {{ site.data.spells.ff }}, еси выбран этот талант.

Всегда поддерживайте {{ site.data.spells.eshield }} на танке и обновляйте его по спадению. В качестве внешнего сейва можно использовать {{ site.data.spells.slt }} – помимо снижения получаемого урона на **10%**, это единственный доступный вариант мгновенно поднять здоровье цели для Шамана Исцеления.

Ваша основная «ротация» для исцеления в подземельях представляет собой следующий порядок заклинаний:
* {{ site.data.spells.riptide }}
* {{ site.data.spells.hs }} для поглощения стака {{ site.data.spells.tw22 }}
* {{ site.data.spells.hw }}
* Любое другое заклинание;

# Скрытые механики способностей

## Цепное исцеление

{{ site.data.spells.ch }} не такое простое заклинание, как может показаться на первый взгляд.

При его использовании считаются бонусы от Универсальности и {{ site.data.spells.ul }}, и они работают на все последующие прыжки. Однако Искусность, Критический удар, {{ site.data.spells.eshield }} и {{ site.data.spells.deluge }} считаются отдельно для каждой цели – то есть применение способности {{ site.data.spells.ch }} на цель под {{ site.data.spells.eshield }} увеличит исцеление на **20%** только по этой цели, но не на все последующие. 

Каждый следующий перескок снижает свою эффективность на **30%**, но это не происходит при активном баффе от таланта {{ site.data.spells.high_tide }}.

Так как мы не можем контролировать прыжки способности {{ site.data.spells.ch }}, очень важно правильно выбрать для него первую цель. {{ site.data.spells.ch }} это полностью **«Смарт хил»**, то есть оно не просто выбирает раненую цель, а **самую раненую**. Большинство заклинаний с обычным смарт хилом выбирают случайную раненую цель, не обращая внимания на количество здоровья, в отличие от {{ site.data.spells.ch }}.

Определяя свой маршрут, {{ site.data.spells.ch }} отдает приоритет тем, у кого не хватает наибольшего количества здоровья численно, а не процентно.

**Пример:** если у одного игрока остался **1%** здоровья, а до **100%** не хватает **4000** хп, а у второго осталось **5%**, но до **100%** не хватает **5000** хп, то {{ site.data.spells.ch }}, выбирая между ними, похилит в первую очередь, второго. 

{{ site.data.spells.ch }} всегда пытается найти самые эффективные цели для перескока. Например, оно может перепрыгнуть на здоровых игроков или питомцев ради того, чтобы добраться до наиболее раненых.

## Родник

{{ site.data.spells.wellspring }}, талант седьмого тира, имеет уникальную механику – это не просто АоЕ отхил, но он к тому-же перемещается в пространстве и хилит всех, кого заденет. Максимальную пользу эта способность нанесет при попадании по **6** целям или более, а если целей будет меньше, то общий отхил будет слабее.

Отхил между всеми целями после **6** распределяется поровну, как и у любой другой лечащей способности с софт-капом целей (подробнее про софт-кап можно почитать [в этой статье](https://stormkeeper.ru/info/target_cap.html)). Главное что следует понять – **не старайтесь задеть этой способностью как можно больше раненых союзников, главное задеть хотя бы шестерых.**

Однако, у таланта {{ site.data.spells.wellspring }} есть одно важное отличие – все избыточное исцеление по целям со **100%** здоровья снижается **до 25%**, чтобы не возникало несбалансированных ситуаций с талантом {{ site.data.spells.cbt }}. Это можно увидеть на следующем графике:

## Щит земли

* Бафф на исцеление действует только на того шамана, кто наложил {{ site.data.spells.eshield }}.
* Исцеление от тотемов и аксессуаров **не усиляется** на **20%**.
* Эффекты периодического исцеления проверяют наличие {{ site.data.spells.eshield }} каждый раз при лечении, а не в момент наложения.
* Исцеление от {{ site.data.spells.eshield }} не усиливается само от себя.
* Внутренний кулдаун на лечение при получении урона – **3** секунды.
* {{ site.data.spells.eshield }} от другого шамана никак не взаимодействует с вашим. Они могут срабатывать как вместе, так и по отдельности.
